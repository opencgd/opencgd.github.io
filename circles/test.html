Point at a circle to zoom in and enter text, use the left mouse button to go backwards.
The zoom speed depends on how close to a circle you point

Limitations: Only represents a few hours work so the sorry the max speed isn't adjustable,
and the program only uses the last 3 letters to predict the next letter.

Fractal circles text entry program by Andrew Gower
Inspired by 'Dasher' by David Ward
See: http://wol.ra.phy.cam.ac.uk/djw30/dasher/
