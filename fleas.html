Unable to start java applet

Either your browser doesn't support java or you have java disabled.

Troubleshooting

  • To play this game you need to have a java capable browser, such as Netscape 4 or Internet Explorer 4 or 5.
  • If the applet still doesn't appear you may have java disabled on your browser. Go to the options/preferences menu and ensure java is enabled.

Quick Instructions
Use the 3 block shapes to help guide your fleas from the entrance to the exit.

To place a block on the map, first click on the block you would like to place at the bottom of the screen, then click on the map where you would like to add the block.

Each level has a password to save you replaying levels you have already won.
