Suggestion discussion board
We are always interested to know how you think we should improve the game. If you have a good idea then post it here. Simple ideas which can be added quickly are the most likely to be added. Suggestions should be explained properly. For instance - if you post a message asking for more monsters or quests make sure you specify what sort of monsters you want, and what the quests should involve.
